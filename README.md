## 🎯 Overview

This project is a **modern AI chat interface inspired by Gemini**, designed with a clean UI and responsive layout.

It is built to be easily integrated with real AI APIs like OpenAI, Gemini, or Groq.

---

## 🚀 Features

* 💬 Modern AI chat interface
* 🔐 Login & Registration system
* 📱 Responsive design
* 🎨 Clean and minimal UI
* ⚡ Ready for backend integration

---

## 🧠 Purpose

This project demonstrates how to build a **frontend interface for AI systems**, focusing on usability and user experience.

* CSS3
* JavaScript
